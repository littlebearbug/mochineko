> **⚠️ 风险提示**：修改游戏内存数据存在一定风险（如存档损坏或账号异常）。请务必在操作前**备份存档**，建议先在小号上测试成功后再操作主号。

## 🛠️ 准备工作

在开始之前，请确保你已经准备好以下环境：

1. **修改工具**：Cheat Engine (CE) 任意版本。
2. **游戏环境**：造梦西游微端（推荐）或浏览器端。
3. **游戏道具**：背包中需要有一把普通的【**月牙铲】**（作为被替换的耗材）。

---

## 📝 核心代码

我们需要用到两串代码，一串是你要修改的目标（头衔），另一串是你背包里现有的物品（铲子）。

### 1. 目标代码（白骨清明头衔）

这是我们最终想要得到的物品数据：

```text
1||bgqmtx|zbtx||||1e5|1e5|1e7|1e8|1|1|1e5|1e5|false|false|false|||||1||10|100||0
```

### 2. 原始代码模板（月牙铲）

这是你需要根据自己装备数值进行微调的搜索代码：

```text
1||ptdyyc|zbwq|0|||0|0|攻击力|0|0|0|0|0|false|false|false|false|false|0|0|1|0|0|0|0|0
```

## 🚀 详细操作流程

### 第一步：构造专属搜索代码

由于每把武器的属性不同，我们需要先“定位”你背包里的那把铲子。

1. 打开游戏，把鼠标移到背包里的【月牙铲】上。
2. 记下它的**攻击力**数值（例如：攻击力是 `4`）。
3. 将上方**原始代码模板**中的文字 `攻击力` 替换为你实际的数值。
   - *修改前*：`...|0|0|攻击力|0|0...`
   - *修改后*：`...|0|0|4|0|0...`
4. **复制**这串修改好的代码。

### 第二步：CE设置与进程连接

1. 打开 Cheat Engine。
2. 点击左上角的“小电脑/放大镜”图标，选择**造梦西游的进程**并打开（微端通常显示为造梦西游图标，浏览器需尝试Flash或Plugin进程）。
3. 在主界面的 **Value Type (数值类型)** 下拉菜单中，选择 **String (字符串)**。

### 第三步：开启“时间暂停” (卡保存关键)

这是最关键的一步，利用变速齿轮延长数据上传的时间窗口。

1. 在 CE 主界面右侧，勾选 **Enable Speedhack (启用变速齿轮)**。
2. 将输入框中的速度值修改为 **0.001**。
3. 点击 **Apply (应用)**。
   - *现象：此时你会发现游戏画面变得极度缓慢，几乎静止。*
4. 回到游戏，点击 **【保存游戏】** 按钮。
   - *现象：由于网速被“物理降速”，屏幕上不会立刻弹出“保存成功”的提示，这正是我们需要的修改时机。*

### 第四步：扫描与替换数据

趁着游戏正在努力“上传保存数据”的间隙，我们进行偷梁换柱。

1. **扫描**：在 CE 搜索框粘贴 **第一步构造好的月牙铲代码**，点击 `First Scan` (首次扫描)。
2. **提取**：扫描结果出来后（通常会有几个地址），按 `Ctrl+A` 全选左侧结果，将其拖入下方的编辑栏。
3. **修改**：
   - 在下方编辑栏再次 `Ctrl+A` 全选所有地址。
   - 双击任意一行数值部分（Value），弹出修改框。
   - 将内容全部删除，粘贴 **目标代码（白骨清明头衔）**。
   - 点击确定。

### 第五步：恢复时间与生效

1. 修改完毕后，不要着急。回到 CE 的变速齿轮设置。
2. 将速度值改回 **1**，点击 **Apply (应用)**。
3. **见证奇迹**：回到游戏画面，你会看到刚才点击的 **“保存成功”** 提示框此刻才弹出来。
4. 点击确定，然后**返回主菜单**（或直接重启游戏）。
5. 重新读取存档进入游戏，打开背包，你原来的月牙铲已经变成了属性逆天的【白骨清明头衔】！

## ❓ 常见问题 (FAQ)

**Q: 扫描不到数据怎么办**？A: 只有两种可能。一是你没有选择正确的游戏进程（特别是浏览器端）；二是你的月牙铲攻击力数值填错了，请仔细核对代码模板中的位置。

**Q: 为什么修改完游戏崩溃了**？A: 代码复制不完整或者格式错误会导致数据溢出。请确保复制的代码没有任何多余的空格或换行。

**Q: 其他装备可以这样改吗**？A: 原理是通用的。只要你有对应物品的原始代码和目标代码，利用“卡保存法”都可以实现替换。
